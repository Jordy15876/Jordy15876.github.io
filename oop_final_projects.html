    This page is where I have placed my assignments that were due at the end of Unit 7 and 11. These two assignments cover System Design and System Implementation.

    System Design At the end of Unit 7, I submitted my System Design for my driverless car program. This design had the requirements of designing a system for a driverless car. I was tasked with identifying three (3) key operations that my software would support and provide the capability for these operatioins. In this design I was also required to create UML diagrams that showed how the program would operate. This required me to conduct extensive research on how driverless cars operate and also on how to properly craft the various diagrams. Tutor Feedback
    System Implementation At the end of Unit 11, I submitted my System Implementation project. This required the implementation of the system design into a program for a driverless car. In this program I was able to implement some of the Obejct-Oriented Programming concepts that I had been learning throughout the module. The program that I wrote was able to do most of what I had set it out to do and I made every attempt to follow what my UML diagrams were supposed to model. Tutor Feedback
